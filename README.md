🎮 Brick Break Arena

 Overview
Brick Break Arena is a 2D arcade-style game developed using Java Swing and AWT. The game demonstrates real-time gameplay where the player controls a paddle to bounce a ball and break bricks.

This project focuses on core programming concepts such as object-oriented programming, event handling, collision detection, and graphical rendering.

 Objectives
- Implement a real-time interactive 2D game
- Apply Java Swing and AWT for GUI development
- Handle keyboard-based user input
- Implement collision detection logic
- Develop scoring and game state system

 Features
- Paddle control using keyboard (Left & Right keys)
- Real-time ball movement
- Brick breaking mechanism
- Dynamic score tracking
- Game Over and Win conditions
- Restart functionality

 Technologies Used
- Java (JDK 8+)
- Java Swing
- Java AWT
- VS Code

System Requirements
- OS: Windows / Linux / macOS
- RAM: Minimum 4GB
- Processor: Intel i3 or above
- JDK installed
